Invoke the Grove colony for thorough research and discovery.

## Process

1. **Define the Question**
   - What do we need to understand?
   - What decisions depend on this research?
   - What's the scope of exploration?

2. **Cast Wide Net**
   - Search documentation
   - Explore codebase
   - Consult external resources
   - Look in unexpected places

3. **Analyze Findings**
   - Evaluate source quality
   - Compare approaches
   - Identify patterns

4. **Synthesize Insights**
   - Connect findings to project needs
   - Draw actionable conclusions
   - Note uncertainties

5. **Document for Team**
   - Create shareable summary
   - Include source references
   - Suggest next steps

## Usage

```
/explore [topic or question]
```

## Example

```
/explore Best practices for WebXR hand tracking implementation
```
