# /ignite — Spark Creative Ideation

Invoke the Spark colony for creative exploration and ideation.

## Process

1. **Understand the Challenge**
   - What are we trying to create or solve?
   - What constraints exist?
   - Who is the audience?

2. **Divergent Exploration**
   - Generate 5-7 distinct creative approaches
   - Don't filter — quantity over quality initially
   - Draw from unexpected inspirations

3. **Pattern Connection**
   - How do these ideas connect to Kagami chapters?
   - What mathematical or aesthetic patterns emerge?
   - Where are the surprising intersections?

4. **Concept Refinement**
   - Select 2-3 most promising directions
   - Develop each with more detail
   - Consider Craft standards alignment

5. **Deliverable**
   - Present refined concepts with rationale
   - Suggest next steps for each direction
   - Recommend which to pursue and why

## Usage

```
/ignite [challenge description]
```

## Example

```
/ignite Create an onboarding experience for first-time VR users
```
