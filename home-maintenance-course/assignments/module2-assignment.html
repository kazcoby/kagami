Assignment: HVAC Filter Change Documentation

Overview

This hands-on assignment guides you through locating, inspecting, and (if needed) changing your HVAC filter. You'll document the process and create a reminder system for future changes.

Time Required: 30-45 minutes total

Instructions

Part 1: Filter Documentation (40 points)

Locate your HVAC filter and document the following with photos and notes:

  1. Filter location photo — Show where the filter is located in your home (10 pts)
  2. Filter size — Read the size from the filter frame (e.g., 16×25×1) (10 pts)
  3. Current condition — Photo of the filter showing cleanliness/dirtiness (10 pts)
  4. Airflow direction — Note which way the arrow points when installed correctly (10 pts)

Part 2: Filter Change OR Inspection Report (30 points)

Option A: If your filter needs changing:

  • Purchase the correct replacement
  • Document the change with before/after photos
  • Write the date on the new filter
  • Note where you purchased the filter and cost

Option B: If your filter is still clean:

  • Explain why you believe it doesn't need changing yet
  • Note when it was last changed (if known)
  • Describe the filter type and its typical lifespan
  • Indicate when you plan to check again
Renters: If you cannot access your filter or are unsure about changing it, document where you looked, and include a copy of your maintenance request to your landlord asking about filter changes.

Part 3: Reminder System (30 points)

Create a system to remind yourself about filter changes:

  • Set up a recurring calendar reminder (phone, Google Calendar, etc.)
  • Include a screenshot of your reminder setup
  • Explain your reminder interval (why monthly, bi-monthly, quarterly?)
  • Bonus: Stock up—note if you bought multiple filters for convenience

Submission Requirements

  • Submit all photos in a single document (PDF preferred) OR as labeled image files
  • Include written explanations for each part
  • Screenshot of your reminder system

Grading Rubric

Criteria Excellent (100%) Satisfactory (75%) Needs Improvement (50%) Points

Filter Change/Report Complete documentation of change OR thorough inspection report Adequate documentation with minor gaps Incomplete or unclear documentation 30

Total Points: 100
