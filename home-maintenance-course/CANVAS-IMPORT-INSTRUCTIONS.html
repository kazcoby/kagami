Canvas Import Instructions

Course: Basic Home Maintenance for Renters (and New Homeowners)

Version: 1.0

Canvas Instance: ccconlineed.instructure.com

Course Contents Overview

This course package includes:

  • 4 Modules with 6 pages each (24 content pages total)
  • 4 Discussions (one per module)
  • 4 Assignments with rubrics
  • 4 Quizzes with answer keys (29 questions total)
  • 8 Downloadable Templates

Option 1: Manual Import (Recommended)

Since Canvas import from Common Cartridge can vary by institution settings, the most reliable method is manual content creation using these files as source material.

1Create a New Course in Canvas

  • Go to your Canvas dashboard
  • Click "+ New Course" (or ask your admin to create one)
  • Name it: "Basic Home Maintenance for Renters (and New Homeowners)"

2Create Module Structure

  • Go to Modules in your new course
  • Create 5 modules:
    1. Course Overview
    2. Module 1: Home Orientation + Safety
    3. Module 2: Air, Appliances + Efficiency
    4. Module 3: Water + Moisture
    5. Module 4: Simple Repairs + Maintenance System

3Add Content Pages

For each HTML file in the modules/ folder:

  • Create a new Page in Canvas
  • Open the HTML file in a browser
  • Copy the content (Ctrl+A, Ctrl+C)
  • Paste into Canvas Rich Content Editor
  • Or: Switch to HTML view and paste the content between <body> tags

4Create Discussions

For each file in discussions/:

  • Create a new Discussion in the appropriate module
  • Copy the prompt content from the HTML file
  • Set points (10 points each)
  • Enable threaded replies

5Create Assignments

For each file in assignments/:

  • Create a new Assignment
  • Copy instructions and rubric from the HTML file
  • Set points as specified (15-30 points)
  • Set submission type (File upload or Text entry)
  • Create rubric based on the table in each file

6Create Quizzes

For each file in quizzes/:

  • Create a new Quiz
  • Add questions (Multiple Choice) from the file
  • Set correct answers from the Answer Key section
  • Configure: 2 attempts, highest score, show answers after submission
  • Add time limit (15-20 minutes)

7Add Templates as Files

  • Go to Files in your course
  • Create a folder called "Downloadable Templates"
  • Upload all files from the templates/ folder
  • Create a Page linking to these files, or add directly to a Module

Option 2: Common Cartridge Import

If your Canvas instance supports Common Cartridge imports:

1Create the .imscc Package

From Terminal, navigate to the course folder and create a ZIP:

cd ~/Desktop/robots/home-maintenance-course
zip -r home-maintenance-course.imscc .

2Import in Canvas

  • Go to Settings > Import Course Content
  • Select "Common Cartridge 1.x Package"
  • Upload the .imscc file
  • Select what to import
  • Click Import
Note: Common Cartridge imports may require cleanup. The HTML content imports well, but discussions, assignments, and quizzes may need to be recreated manually for full functionality.

Accessibility Reminder:

All content has been designed with accessibility in mind:

  • Proper heading structure (h1, h2, h3)
  • High contrast colors
  • Clear, readable fonts
  • Text alternatives for visual elements (DIY vs Pro Decision Tree)
  • Structured tables with headers
